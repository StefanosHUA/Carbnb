# Carbnb

A simple React app to browse and rent cars.

## Getting Started

1. **Install dependencies:**
   ```
   npm install
   ```
2. **Start the development server:**
   ```
   npm start
   ```
   This will open the app at http://localhost:3000

3. **Build for production:**
   ```
   npm run build
   ```

## Project Structure
- `src/` - React source code
- `public/` - Static files

## Requirements
- Node.js >= 14
- npm >= 6
